## 博客
指令来源:[Hexo](https://hexo.io/zh-cn/docs/commands)
### 创建博客

```cmd
hexo init [folder]
cd <folder>
npm install
```
### 文章与页面
`hexo new page [name]`创建新页面

`hexo new post [name]`创建新文章
### 文件与上传
`hexo generate` 生成静态文件

`hexo server` 启动服务

`hexo deploy` 部署网站

`hexo clean` 清除缓存文件 (db.json) 和已生成的静态文件 (public)
## 插件
### hexo-tag-steamgame
